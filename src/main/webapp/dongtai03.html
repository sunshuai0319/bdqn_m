青鸟java毕业师哥收到offer，曝光简历和自我介绍手稿

王豪在分享会

有一位师哥名叫王豪，是北大青鸟学士后java JT09班一位马上面临毕业的学员。在结课前一周，他开始紧锣密鼓投递简历，一个星期内接到了9家企业的面试机会，面试后收到了7家企业发给他的“offer”！

毕业季在很多学员为了成功就业抓耳挠腮的时候，这位师哥究竟用了什么高明的方法，成功概率如此之高？小青分享过很多毕业学员面试时我们应该注意的问题，学长学姐也总结出了答案。但是还没有毕业学员分享过他们面试时的简历到底是如何写的？自我介绍时要重点突出什么？今天小青请到了这位师哥，他到底是如何找到心仪工作的（小青把他应聘时的简历和自我介绍手稿都要来了！）？我们听一听他的分享：

1 面试前的准备——专业技能过硬

零基础学习Java编程，我给大家分享的宝典是“永远要走在老师的前头，千万不要让老师领着你去赛跑”。平时大家学习应该用好云课堂和云题库。云课堂上的学习要赶在面授课前完成，每次看完云课堂通过云题库做评测，了解自己撑握的情况。因为时间是不管你花与不花都会过，所以我想我们应该花在值得的事情上。以下是具体方法：

1、“利用预科学习阶段，提前进行云课堂的预习。

2、专业课开课除了每天上正课和总部直播课外，就是对课后的做作业要多练习，个人心得：不理解先搞一遍两遍，更多的时候会更好理解。

3、晚上的自习一定要坚持在校实训，在校实训效率高，有同学一起讨论，有老师请教。减少了在家通过网络提问，时效性差，通常耐性会在等待中消磨掉。

4、敲代码多记英文单词，对于老师要求记的一定要寄熟悉，其实不多也就50多个常用的。

2 面试过程——经验过硬

面试第一步——笔试

笔试是技术人员面试第一步，只有顺利通过笔试才能进入面试。我面试经验得出做JAVA工程师面试以基础题为主，SQL，框架为次；另外很多企业会关注我们对行业新技术和相关技术的了解比如web service，Extjs，所以平时拓宽专业自己知识面也是重要的。以下是我提供给大家几个重要知识点：比如java实现字符串反转，常用集合的适用场合（ArrayList、LinkedList、Vector）。

以下是我的参考答案：

1、 java实现字符串反转：

public void changestr(string s){

char[ ] c=s.toCharArray();

for(int i=c.length-1;i≥0;i--){

system.out.println(c[i]);

}

}

2、常用集合的适用场合（ArrayList、LinkedList、Vector）

ArrayList：类似于数组存储数据方式，循环遍历数据效率高；

LinkedList：采用链表式结构存储数据，对数据的增删改效率高；

Vector：采用线程同步的存储方式，操作数据效率低。 

面试第二步——面试

通过第一轮笔试，可以说明我们技术能力是已经得到企业的肯定的，那如何能更好地表现自己呢？有一个好的自我介绍是自己谈薪资的前提。这需要我们有前期的准备，就业前就业老师和教员给过我共4次深入的指导和修改，通过模拟面试，让我知道问题的原由。我就自我介绍还做了一份文稿，它是我们简历的诠释。（开始有点小难，因为我是工科男)。就业指导老师修改了，通过了，默默熟背下来，接着就是演练了。

王豪自我介绍的手稿

这一周我去了九家企业面试，其实每一次的面试都是一机会：面试前了解企业信息和招聘岗位的信息，提前查询乘车路线。每一次面试后不管成功与否，自己都会做一个总结分析，记录面试过程中问到的问题，这就为下一次面试积攒了经验。试想，如果上一家企业面试你java实现字符串反转的知识点，你回答得不完美或者模棱两可。如果你回去掌握了碰巧另一家面试问到同样的知识点，你对答如流，面试官想拒绝你都难啊~

3 过硬的装备

面试过程中保持自信，在面试过程中，要时刻注意面试官的表情，不管遇到什么问题，会或者不会，保持微笑都能给人好感觉，可以加分。

正装出场也很重要：让企业知道我们专门为了面试而做了准备，让企业人事知道我们重视公司给我们的这次面试机会。

面试后也要有耐心，有准备的去做事。多做总结找原因，与就业老师交流。等待的焦急开始会有，但不要因为面试不成功就失去自信，遇到合适的企业也不敢谈待遇，只为求工作。

4 什么是你理想的职业

王豪的个人简历

我过完年后，知道即将面临就业，那时和班里的其他同学一起沟通都有点担心。也想知道什么的职业才是自己理想的职业，在大家一次次学习讨论中，我觉得：”理想的职业应该是自己喜欢、自己能干、社会需要、前景广阔。从长远看，越来越“增值”——也就是经验、价值会随着经历的增长而更快增长。有满足这些要求的才能说是一个好的职业。现在我的想法就是：还好我没有错过互联网，如果错过了，我想与我们擦肩而过的应该不仅是就业机会而是整整一个时代。

王豪在学校，学习努力，还常常跟班里同学一起做分享交流讲试题，辅导后进小伙伴。他说这样自己学得确实扎实，说不信你们也试试~他还积极参加学校大小活动，增加人脉历练胆识。他已正式上班，他说转正后工资向1w冲刺，试用期挣6500还是太少了！
